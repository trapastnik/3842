# CLAUDE.md

Репозиторий с прототипами интерактивных арт-объектов для пяти мультимедийных комплексов (МТК) музея В.И. Ленина: **МТК 38, 39, 40, 41, 42**.

## Многосессионный режим

Проект ведётся в **параллельных Claude-сессиях**, по одной на каждый МТК. Перед началом любой работы:

1. **Прочитай [SETUP.md](SETUP.md)** — пошаговая инструкция для новой сессии.
2. **Узнай у пользователя, над каким МТК ты работаешь** (38 / 39 / 40 / 41 / 42).
3. Соблюдай файловые границы из [COORDINATION.md](COORDINATION.md). Никогда не правь файлы чужого МТК.
4. Архитектура и план — [PLAN.md](PLAN.md).

## Ключевые правила (резюме)

- Сессия по МТК `XX` владеет: `mtkXX-*/`, `data/mtkXX.json`, секциями про свой МТК в README/index.html.
- Не трогай `mtkYY-*/` или `data/mtkYY.json` для других МТК.
- В `README.md` и `index.html` правь только свою секцию.
- Коммить в свою feature-ветку (`claude/<worktree-name>`), не в `main`.
- Общий HTTP-сервер один — на порту **8092** из корня репозитория.
- Если меняешь общие правила или фиксируешь хронологию слияний — пиши в [COORDINATION.md](COORDINATION.md).

## Стек

Vanilla JS, без бандлеров. **WebGL разрешён** (решение 2026-06-10) — для арт-качества на 4K, во всех МТК, при **одном жёстком условии: все библиотеки вендорятся локально** (Three.js, regl, pixi и т.п. — бандл в `<prototype>/vendor/` или `assets/shared/vendor/`, без рантайм-CDN). Музейный киоск может стоять офлайн — любая зависимость от сети = чёрный экран. Canvas 2D остаётся валидным выбором там, где WebGL избыточен.

Шрифты: бренд — `mtk38-globe/fonts/` (Nolde, 21 Cent, 20 Kopeek, покрывают латиницу+кириллицу); не-латиница (29 письменностей Noto: CJK/Arabic/Indic/…) — `assets/shared/fonts/noto/` (bundled, SIL OFL; список — `mtk38-handoff/FONTS-NEEDED.md`). Брендовая палитра — см. `COORDINATION.md` → «Шрифтово-цветовые правила» (актуальная редакция от 2026-05-12).

Внешние зависимости — **только вендоренные** (см. `COORDINATION.md` → «Внешние зависимости»). На миграцию остаётся `pretext` через esm.sh в `mtk38-poster` — задача владельца МТК 38.
