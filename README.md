Промежуточные материалы и прототипы для интерактивных арт-объектов МТК 38-42.

- `МТК №38...docx` — исходное ТЗ для арт-объекта «Ленин на языках мира».
- `МТК №39...docx` — исходное ТЗ для арт-объекта «Имени Ленина».

- `МТК №42...docx` — исходное ТЗ для арт-объекта «Осмысление наследия Ленина».
- `mtk38-globe/` — текущий интерактивный canvas-прототип для МТК 38.

## МТК 38 · Ленин на языках мира

Прототип исследует центральную визуальную метафору: шар из написаний слова «Ленин» на разных языках и письменностях. Основа — 49-дюймовый вертикальный тач-экран, стилистически привязанный к брендбуку Ленин-центра: темный графитовый фон, латунные акценты, красная диагональ, крупная типографика.

### Запуск

```bash
cd mtk38-globe
python3 -m http.server 8092
```

Открыть: http://127.0.0.1:8092/

Можно открыть сразу режим сферы:

```text
http://127.0.0.1:8092/?mode=sphere
```

### Режимы

`Слои` — основной текущий вариант. Слова расположены по параллелям; свайп по ближайшей параллели вращает только этот слой. Плотность надписей рассчитывается по фактической ширине слов, чтобы соседние надписи не наезжали друг на друга по горизонтали.

`Сфера` — экспериментальный вариант. Свайп вращает весь шар как поверхность, при этом слова продолжают двигаться по параллелям. Это не WebGL, а 3D-проекция внутри canvas, поэтому вариант остается легким и управляемым.

### Тач-интерфейс

- Hover-состояния не используются.
- Нет реакции на простое движение курсора без касания.
- Визуальный фидбек появляется только во время активного касания/перетаскивания.
- Переключатель режимов сделан крупными тач-кнопками.

### Разработка

- Рабочая версия хранится в `mtk38-globe/`.
- Версии фиксируем git-коммитами, а не ручными копиями папок.
- `versions/` оставлен только как локальный архив экспериментов и игнорируется git.
- Технические скриншоты создаем только временно для проверки, смотрим и сразу удаляем.
- Скриншоты и растровые превью не добавляем в репозиторий.
- Служебные папки `.claude/`, `.DS_Store` и временные файлы Word игнорируются.
