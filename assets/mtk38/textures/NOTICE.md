# Earth-текстуры (equirectangular) — вендорено локально для офлайн-киоска

- `earth-physical.jpg` — natural-color физическая карта (моря/материки/льды), 4096×2048.
- `earth-relief.jpg` — grayscale карта высот/рельефа (горы), 4096×2048.

Источник: производные NASA **Blue Marble** (общественное достояние) через проект
webgl-earth (turban, github). Для музейной инсталляции допустимо. Рантайм-CDN нет —
файлы локальные. Используются в `mtk38-v3/scenes/earth.js` (варианты «рельеф»/«физ.»).
