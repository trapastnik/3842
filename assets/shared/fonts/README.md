# assets/shared/fonts — общие шрифты проекта

Зона координатора. Сюда кладутся шрифты, нужные нескольким МТК (в первую очередь не-латиница для МТК 38).

## Структура

```
assets/shared/fonts/
└── noto/        ← Noto Sans для не-латинских письменностей (bundled, SIL OFL)
                    напр. NotoSansCJK, NotoSansArabic, NotoSansDevanagari,
                    NotoSansGujarati, NotoSansTibetan, NotoSansOlChiki,
                    NotoSansThaana, NotoSansTifinagh, NotoSansNKo …
```

## Правила

- **Бренд-шрифты (Nolde, 21 Cent, 20 Kopeek) сюда НЕ переезжают** — они остаются в
  `mtk38-globe/fonts/`, потому что на относительный путь `../mtk38-globe/fonts/`
  завязаны ~20 прототипов всех пяти МТК. Переезд сломает их разом.
- Noto-шрифты — только те письменности, что реально нужны утверждённому списку языков.
  Каждый файл крупный (часто 5–15 МБ), лишнее не кладём.
- Лицензия Noto — SIL Open Font License, коммитится в git (`.gitignore` блокирует
  только изображения, не шрифты).
- Подключение из прототипа: `@font-face { src: url("../assets/shared/fonts/noto/<file>"); }`.

## Статус

Каркас заведён 2026-06-10. Canonical-список МТК 38 v2 = **52 языка** (на листе PDF 52
надписи: 50 подписанных + русская/латинская опоры; 53-го нет). Сессия МТК 38 вывела
из `data/mtk38.json` список **29 не-латинских письменностей**, которым нужен bundled
Noto — см. `mtk38-handoff/FONTS-NEEDED.md` (с рекомендованными файлами и ISO-кодами).
Бренд-шрифты покрывают латиницу+кириллицу. Файлы Noto координатор кладёт сюда по этому
списку (отдельная задача, ~29 файлов SIL OFL).
