## Фазы

### Ф0 · Данные (фундамент) — ✅ ГОТОВО
- **0a.** `data/mtk38.json` — 52 языка по схеме Р2. Написание «Ленин» — с PDF
  (`writing_source: pdf-specimen`); не-латиница → `verifier: unverified` + флаг
  `needs-verification` на подозрительные глифы (мейтей, тибетский, шанский, тхана,
  нко, лаосский, «дзинка»). Метаданные: эндоним, скрипт ISO 15924, семья, гео
  `primary`+`also` (Р3), диаспора (Р4), speakers, weight.
- **0b.** `data/mtk38-quotes.json` — 10–20 фраз самого Ленина (ПСС/Marxists.org),
  оригинал + переводы на топ-языки, каждая c `source` и `verifier`. Строгая
  верификация против misattribution; спорные/«о Ленине» не берём.

### Ф1 · Шрифты (блокер качества) — ✅ ГОТОВО (subset-вариант)
- `mtk38-v2/fonts/noto/` — 57 woff2, 29 письменностей × (400+700), subset под нужные глифы
  (`build_fonts.py`), SIL OFL. Самодостаточно → рендер на любой машине, офлайн.
- Бренд-шрифты (Nolde, 21 Cent, 20 Kopeek) — в `mtk38-globe/fonts/`.
- WebGL вендорен: `mtk38-v2/vendor/three/` (Three.js r137 UMD + UnrealBloom).
- ⬜ Остаётся долг: вендорить `pretext` → `mtk38-poster/vendor/` (всплывёт на пересборке постера).

### Ф2 · Рендер + 8 вариантов — 🔶 2/8
Базовый пакет на каждый (рецепт — в START-v2.md): структура из v1-прототипа `mtk38-<name>/`;
данные→`mtk38.json`; снять вертикаль (Р1); палитра Р5b; пост-обработка Р6-A
(bloom/grain/виньетка); bundled-шрифты Р6-B; тюнинг-панель (параметры настраиваемы);
карточка/модалка Р2 где есть детали; слой цитат где уместно.

- ✅ **globe** — `globe.html` (`build_globe.py`): сфера из 19 колец, drag+инерция, лимб, тап→карточка.
- ✅ **map** — `map.html` (`build_map.py`): equirectangular, контуры стран (инлайн geojson), точки по гео Р3/Р4, pan+инерция, тап→модалка.
- ⬜ **catalog** — сетка 6×7 карточек Р2, hover/тап→регион.
- ⬜ **mandala** — кольца + медальон-центр, drag кольца с пружиной, красный центр точечно.
- ⬜ **rain** — ~130 частиц anti-gravity, 4 яруса, repulsion пальцем.
- ⬜ **ticker** — ~10 лент, разные скорости/стороны, drag с пружиной.
- ⬜ **poster** — 16 hero-слов в 4 яруса + filler-слои, курсор-препятствие; + вендорить pretext.
- ⬜ **cloud** — 3D-объём, слайдер плотности ×1..20, орбита камеры, дыхание.

Рекомендуемый порядок добивки: catalog → mandala → rain → ticker → poster → cloud
(от простого к сложному). Рендер — canvas-2D + пост по умолчанию; WebGL (вендоренный)
точечно, где даёт скачок на 4K.

> Примечание: `mtk38-v2/app.html` (обобщённый 7-раскладочный движок) — ранний черновик
> ДО коррекции по V1. Не эталон; оставлен как песочница, в зачёт 8 вариантов не идёт.
