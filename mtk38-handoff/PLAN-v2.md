## Фазы

### Ф0 · Данные (фундамент, блокирует Ф1–Ф2)
- **0a.** `data/mtk38.json` — 52 языка по схеме Р2. Написание «Ленин» — с PDF
  (`writing_source: pdf-specimen`); не-латиница → `verifier: unverified` + флаг
  `needs-verification` на подозрительные глифы (мейтей, тибетский, шанский, тхана,
  нко, лаосский, «дзинка»). Метаданные: эндоним, скрипт ISO 15924, семья, гео
  `primary`+`also` (Р3), диаспора (Р4), speakers, weight.
- **0b.** `data/mtk38-quotes.json` — 10–20 фраз самого Ленина (ПСС/Marxists.org),
  оригинал + переводы на топ-языки, каждая c `source` и `verifier`. Строгая
  верификация против misattribution; спорные/«о Ленине» не берём.

### Ф1 · Шрифты (блокер качества)
- Из 0a — список ~30 письменностей → координатору для `assets/shared/fonts/noto/`.
- Бренд-шрифты (Nolde, 21 Cent, 20 Kopeek) остаются в `mtk38-globe/fonts/`.
- Долг: вендорить `pretext` → `mtk38-poster/vendor/pretext.min.js`.

### Ф2 · Рендер + 8 вариантов
Базовый пакет на каждый: данные→`mtk38.json`; снять вертикаль (Р1); палитра Р5b;
пост-обработка Р6-A (bloom/grain/виньетка); bundled-шрифты Р6-B; SVG hero-глифы Р6-C.
Специфика: **map** — гео Р3/Р4 + цитата в модалке; **catalog** — 52 карточки Р2 +
цитата; **poster** — pretext + цитаты-слой; **globe** — сигнатурный 3D + hero-глифы;
**cloud/mandala/rain/ticker** — данные+горизонталь+рендер.
Порядок: globe → map → catalog → poster → cloud → mandala → rain → ticker.
Рендер — canvas-2D + пост-обработка по умолчанию; WebGL (вендоренный) точечно, где
даёт скачок на 4K.

### Ф3 · Полировка + сдача
Кросс-вариантная консистентность, производительность на 3840×2160, прогон всех 8
через хаб в горизонтали, повариантный merge `claude/mtk38-worker` → main.

## Риски
- **Точность написаний:** 52 взяты визуально с image-PDF. Латиница/кириллица надёжны;
  экзотические письменности — флаг «черновик, требует носителя» (виден по `verifier`).
- **Цитаты:** жёсткая верификация; неподтверждённое в данные не попадает.
