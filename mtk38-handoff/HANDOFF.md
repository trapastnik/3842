## Решения с пользователем/координатором (Р1–Р6 — обязательны)

- **Р1 · Формат:** горизонталь only, 3840×2160 (4K, тач 49″). Вертикаль убрать полностью (снять все `@media (orientation: portrait)`).
- **Р2 · Карточка языка:** поля выше. 5 закрываются машинно; «источник написания» + «верификатор» — провенанс, ручная работа.
- **Р3 · Один язык → много стран:** `geo.primary` (акцент + надпись) + `geo.also[]` (приглушённая подсветка без надписи). Не равномерная каша, не «без географии».
- **Р4 · Без территории (идиш и т.п.):** `territorial:false, diaspora:true` → «пояс диаспоры» (орбита/лента по краю), не на земле.
- **Р5 · Цитаты:** короткие фразы **самого Ленина** (не «о Ленине» — это МТК 42) из ПСС/Marxists, каждая с source+verifier. Спорное не показываем. ✅ реализовано в `mtk38-quotes.json`.
- **Р5b · Палитра:** брендбук 2026-05-12 — `--brass #D2B773`, `--paper #F7F9EF`, `--telegrey #CFD0CF`, `--window #9DA3A8`, `--graphite #435059`, `--red #A02128` (красный точечно). Старые токены (`--signal-red`, `--slate-blue`) — убрать.
- **Р6 · Качество «максимально красиво»:** пост-обработка 2D (bloom/grain/виньетка) + bundled-шрифты обязательны; SVG hero-глифы желательны; **WebGL разрешён** (решение координатора 2026-06-10) при жёстком вендоринге библиотек локально (без рантайм-CDN). Canvas-2D валиден там, где WebGL избыточен.
