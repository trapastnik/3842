8 вариантов, каждый — самостоятельная папка `mtk38-<name>/` (index.html + styles.css + один js):

## Решения, принятые с пользователем (обязательны для v2)

### Р1. Формат — горизонталь only
4K тач-панель 49″, **3840×2160, только landscape**. Вертикаль убрать полностью.
**Технический долг к снятию**: убрать все `@media (orientation: portrait)` ветки, добавленные в v1:
- `mtk38-cloud/styles.css` — портретное позиционирование слайдера и крупный thumb;
- `mtk38-ticker/ticker.js` — `isPortrait` font-scale ×1.65;
- `mtk38-map/styles.css` — портретные размеры модалки;
- `mtk38-map/map.js` — портретная развилка вертикального клампа камеры (можно упростить).

### Р2. Структура карточки языка
Поля: **название по-русски · эндоним · письменность · языковая семья · ареал · источник написания · верификатор.**
Машинной триангуляцией закрываются 5 полей; «источник написания» и «верификатор» — провенанс и качество, ручная работа. Схема — в `data/mtk38.json` (draft, см. ниже).

### Р3. Один язык → много стран = «основная + дополнительные»
`geo.primary` (полный акцент + надпись) + `geo.also[]` (приглушённая подсветка без надписи). Не множественная-равная (каша на 4K), не «без привязки» (теряется география).

### Р4. Идиш и языки без территории = категория «диаспора»
`territorial: false, diaspora: true`. На карте — не на земле, а в «поясе диаспоры» (орбита/лента по краю). Сюда же эсперанто, ладино, если появятся.

### Р5. Цитаты — отдельная строгая задача
Для МТК 38 — короткие фразы **самого Ленина** (не о Ленине; «о Ленине» — это МТК 42, чужая зона) из официальных переводов ПСС (Marxists Internet Archive). Малый объём (10–20 топ-языков), каждая — с `source` + `verifier`. Спорные не показываем.

### Р5b. Синхронизировать палитру с обновлённым брендбуком
Дизайнер обновил палитру 2026-05-12 (см. `COORDINATION.md` → «Шрифтово-цветовые правила»): `--amber-black #000`, `--brass #D2B773`, `--red #A02128` (точечный акцент, НЕ заливка), `--blue-grey #5D8970`, `--window #9DA3A8`, `--graphite #435059`. Прототипы v1 используют старые токены (`--signal-red`, `--slate-blue`). При переработке — перейти на актуальные.

### Р6. Качество рендера — «максимально красиво»
Разрешён более художественный рендер. Приоритет: **A (пост-обработка 2D) + B (bundled-шрифты) обязательно**, **C (SVG hero-глифы)** желательно, **D (WebGL)** — ✅ **разрешён координатором проектно 2026-06-10** (см. CLAUDE.md → «Стек»), при жёстком условии: все WebGL-библиотеки (Three.js / regl / pixi …) **вендорятся локально** в `<prototype>/vendor/` или `assets/shared/vendor/`, без рантайм-CDN (киоск офлайн). Canvas 2D остаётся валидным там, где WebGL избыточен.

## Что делать в v2 — план (черновик, требует утверждения)

### Фаза 0. Данные (3 отдельных research-блока)
1. **Canonical-список языков** — название/эндоним/скрипт/семья/ареал. Источники: Wikidata (P1705 эндоним, Q-объекты языков), Glottolog (семья), ISO 15924 (скрипт), Ethnologue (ареал). Машинно, быстро.
2. **Написания «Ленин»** — триангуляция: PDF-specimen ↔ Wikidata Q1394 (label на ~200 языках) ↔ Wikipedia-интервики «Владимир Ленин». Совпало 3/3 → `verified: machine-triangulated`; расхождение → `needs-native-review`.
3. **Цитаты Ленина** (опционально для v2.1) — переводы из ПСС, ручная верификация.

Результат фазы 0 → заполненный `data/mtk38.json` по схеме Р2.

### Фаза 1. Шрифты (блокер качества)
Bundled Noto Sans для всех письменностей списка (особенно новые из PDF: гуджарати, ория, тибетский, манипури/Meitei Mayek, сантали/Ol Chiki, шан, мальдивский/Thaana, нко, тифинаг). Все есть в Google Noto, лицензия SIL OFL. Без них на киоске — квадраты `▯▯▯`. Согласовать с координатором место (`mtk38-globe/fonts/` shared, или новый `assets/shared/fonts/`).

### Фаза 2. Переработка вариантов (горизонталь + новые данные + художественный рендер)
По каждому из 8: перевести на canonical-данные, убрать вертикаль (Р1), добавить пост-обработку (Р6-A). Где есть карточки/детали — структура Р2. Карта — geo-логика Р3 + диаспора Р4.
