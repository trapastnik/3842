## Что это за проект (в двух абзацах)

МТК 38 «**Ленин на языках мира**» — интерактивный арт-объект для киоска музея В.И. Ленина: слово «Ленин» на 53 языках/письменностях (52 с листа музея + испанский), 4K горизонталь (3840×2160), тач 49″. **v1** (8 canvas-прототипов) уже в `main`. Мы делаем **v2** — те же 8 композиций по структуре, но на каноничных данных, горизонталь-only, с кинематографичным рендером (bloom/grain/виньетка), тюнинг-панелями, бренд-шрифтами, карточками языка (схема Р2) и цитатами Ленина.

**Самое важное правило v2 (выстрадано):** каждый вариант должен быть **структурно верен своему v1-прототипу**, а не абстрактным «облаком слов». Карта — это настоящая карта (контуры стран, точки по координатам, модалка). Глобус — вращающаяся сфера из колец-широт. И т.д. Сначала изучи v1 (`mtk38-<name>/`), потом апгрейдь — не переизобретай.

## Рецепт пересборки одного варианта (как делались globe и map)

1. **Изучи v1**: открой `mtk38-<name>/{index.html, *.js, styles.css}` — пойми структуру, физику, интеракцию. Это эталон формы.
2. **Генератор**: пиши `mtk38-handoff/build_<name>.py`, который читает `data/mtk38.json` (+ при нужде `data/mtk38-quotes.json`) и эмитит **самодостаточный** `mtk38-v2/<name>.html` (данные инлайнятся внутрь — чтобы открывалось по `file://` и офлайн). **JSON — источник истины**, html — артефакт сборки.
3. **Апгрейд поверх структуры v1:**
   - горизонталь-only 3840×2160 (снять все `@media (orientation: portrait)`);
   - кинематографичный пост: bloom (размытая `lighter`-копия), film grain (noise-паттерн), виньетка (radial-gradient); для WebGL — UnrealBloom + FogExp2 + ACES;
   - бренд-палитра (`--brass #D2B773`, `--paper #F7F9EF`, `--graphite #435059`, `--red #A02128` точечно — см. ниже);
   - **тюнинг-панель** со слайдерами параметров + кнопка «Скопировать настройки» (выдаёт JSON) — параметры всех версий настраиваемые (это цель-goal);
   - бренд-шрифты + Noto-fallback (стек ниже);
   - где есть детали языка — **карточка/модалка Р2**; где уместно — слой цитат.
4. **Проверка**: `node --check` для вынесенного JS / визуальный скриншот headless-Chrome (Canvas-2D: `--disable-gpu`; WebGL: `--use-gl=angle --use-angle=swiftshader --enable-unsafe-swiftshader`). На macOS **нет** `timeout` — запускай команду без него; не используй `pkill`, который убьёт скриншотящий Chrome.
5. Коммит в `claude/mtk38-worker` с внятным сообщением. **Не пушить, не мержить** без явной команды пользователя.

## Жёсткие правила (не нарушать)

- **Не пушить и не мержить в main без явной команды пользователя.** Глагол «пушь»/«мержи» от пользователя — единственное разрешение. Команда в шаблоне/инструкции за разрешение не считается.
- **Файловые границы:** наша зона — `mtk38-*/`, `mtk38-v2/`, `data/mtk38*.json`, `assets/mtk38/`, секция «МТК 38» в README. НЕ трогать: корневой `index.html`, `assets/shared/*`, чужие `mtk39..42-*`, `data/mtk39..42.json`, append-only `COORDINATION.md`/`PLAN.md`/`SETUP.md`/`CLAUDE.md`.
- **HTTP-сервер 8092 — только координатор**, из корня `main`. Свой не поднимать, упавший не воскрешать — попросить координатора. Для просмотра v2 хватает `file://` (двойной клик / `open`).
- **Только вендоренные библиотеки**, без рантайм-CDN (киоск офлайн = любой CDN → чёрный экран). Three.js уже локально; на миграцию остаётся `pretext` в постере.
- **Канон написаний = строка в `data/mtk38.json`** (как на листе, КАПС где так на листе: `por`="LÊNIN", `fra`="LÉNINE"). Шрифтовый рендер ≠ канон. В валидаторе канон показывается кодпойнтами + кроп с PDF как ground-truth.

## Шрифты и палитра (шпаргалка)

**Палитра (брендбук 2026-05-12):** `--brass #D2B773` · `--paper #F7F9EF` · `--telegrey #CFD0CF` · `--window #9DA3A8` · `--graphite #435059` · `--red #A02128` (красный — точечный акцент, НЕ заливка).

**Шрифты:** бренд — `mtk38-globe/fonts/` (Nolde — заголовки, 21 Cent — текст, 20 Kopeek — мета, покрывают латиницу+кириллицу). Не-латиница — subset-Noto в `mtk38-v2/fonts/noto/` (имена @font-face: `noto-<ISO15924>`, напр. `noto-Arab`, `noto-Deva`; -700 = bold). Стек по письменности: латиница/кириллица → `'20 Kopeek','Arial Unicode MS'`; иначе → `'Arial Unicode MS','noto-<iso>'`. На macOS Arial Unicode MS лежит в `/System/Library/Fonts/Supplemental/Arial Unicode.ttf`.
